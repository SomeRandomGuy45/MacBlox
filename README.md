# Building
### Libraries needed:
* libcurl [Website](https://curl.se/libcurl/)
* curlpp [Github](https://github.com/jpbarrette/curlpp/)
* wxwidgets [Github](https://github.com/wxWidgets/wxWidgets)
### Building the project:
Just run the following command
``
make
``
